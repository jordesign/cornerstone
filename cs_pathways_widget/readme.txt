== Description ==

Cornerstone Pathways makes it easy for you to insert pathway links (a single link around a title, image and text). These can be useful for simple advertising spots in the sidebar, or even to provide access to specific site sections in a widgetised homepage.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

You can add Pathways in the Appearance > Widgets menu
